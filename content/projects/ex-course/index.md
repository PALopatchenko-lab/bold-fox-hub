Этот курс стал для меня первой серьёзной ступенькой в мир Linux. До этого я слышала о Linux только как об «ОС для программистов и серверов». Курс дал именно ту базу, с которой можно уверенно двигаться дальше.

## Что я узнала и чему научилась

### 1. Архитектура и философия Linux
- Поняла, чем Linux отличается от Windows и macOS
- Узнала про ядро, оболочки (shell) и пользовательское пространство
- Осознала, почему «всё есть файл» — это не просто красивая фраза

### 2. Работа в командной строке (Bash)
Вот список команд, которые теперь для меня — как родные:

| Команда | Назначение |
|---------|------------|
| `ls`, `cd`, `pwd` | Навигация по файловой системе |
| `cp`, `mv`, `rm` | Копирование, перемещение, удаление |
| `cat`, `less`, `head`, `tail` | Просмотр содержимого файлов |
| `grep` | Поиск внутри файлов |
| `chmod`, `chown` | Управление правами доступа |
| `ps`, `top`, `kill` | Управление процессами |
| `man` | Лучший друг — встроенная справка |

### 3. Файловая система Linux
- Иерархия директорий: `/bin`, `/etc`, `/home`, `/var`, `/tmp`
- Монтирование устройств
- Символические и жёсткие ссылки

### 4. Управление пользователями и правами
- Как работают `rwx` (чтение, запись, выполнение)
- Суперпользователь `sudo`
- Группы пользователей

### 5. Процессы и окружение
- Как посмотреть запущенные процессы
- Фоновый и передний план
- Переменные окружения (`PATH`, `HOME` и другие)
